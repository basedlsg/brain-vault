## Captures this week (last 7 days)

```dataview
TABLE WITHOUT ID
  file.link AS "Note",
  source AS "Source",
  type AS "Type",
  dateformat(file.ctime, "yyyy-MM-dd HH:mm") AS "Captured"
FROM "inbox" OR "notes" OR "ideas"
WHERE file.ctime >= date(today) - dur(7 days)
SORT file.ctime DESC
LIMIT 30
```
